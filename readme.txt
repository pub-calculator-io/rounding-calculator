This rounding calculator rounds off numbers to the nearest whole number, significant digit, or decimal place. You can round the decimal numbers to the nearest tenths, hundredths, or thousandths.

[https://www.calculator.io/rounding-calculator/](https://www.calculator.io/rounding-calculator/)

== Usage ==

Add the Rounding Calculator shortcode to your page, post or sidebar:

`[ci_rounding_calculator]`

Add the following code to your template where you would like the Rounding Calculator to appear:

`<?php display_ci_rounding_calculator(); ?>`

== Installation ==

1. Upload the Rounding Calculator /ci_rounding_calculator/ folder to the /wp-content/plugins/ directory.
2. Activate the Rounding Calculator plugin through the "Plugins" menu in WordPress.
